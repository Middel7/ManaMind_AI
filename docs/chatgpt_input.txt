MANAMIND AI — CONTEXTE POUR CHATGPT

Projet : ManaMind AI
Objectif : système de recommandation de cartes Magic: The Gathering (format Commander/EDH) pour améliorer les decks existants et proposer des cartes à ajouter/retirer.

Contexte technique
- Langage : Python 3.12
- Framework web : FastAPI + Uvicorn
- Données : fichiers CSV/CSV decklists dans data/Decklists et data/cards_unique_global.csv
- Base de données : SQLAlchemy / Alembic / PostgreSQL (optionnelle, via .env)
- Gestion des dépendances : uv (et non pip)
- Point d’entrée principal : server.py
- Script de recommandation : src/manamind/recommend_deck_changes.py

Structure importante
- server.py : API FastAPI et interface web
- src/manamind/ : logique métier Python
- data/Decklists/ : datasets de decks par commandant
- outputs/ : recommandations générées
- uploads/ : decklists uploadées par l’utilisateur
- docs/ : documentation projet

Fonctionnement actuel
- L’application permet de charger un decklist, de générer des recommandations CSV, et d’afficher les résultats via un front web.
- Le système s’appuie sur des statistiques de cooccurrence entre cartes et sur les decks similaires par commandant.
- Le projet est prévu pour évoluer vers des recommandations plus avancées (content-based, hybrid, DB enrichie, interface utilisateur améliorée).
